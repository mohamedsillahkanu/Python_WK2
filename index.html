1Ô∏è‚É£ Lists, Arrays, Dictionaries, and DataFrames

What are Data Structures?

üìå Data structures are how data is organized and stored in Python. Understanding these is key to effective data analysis.

Step 1: Lists

# Creating a list of malaria cases per region
cases_per_region = [150, 200, 250, 300]

üîç Explanation:

‚úî [] creates a list‚Äîa basic data structure in Python
‚úî cases_per_region stores the number of malaria cases in four different regions

Step 2: NumPy Arrays

# Creating a NumPy array of malaria cases (rows = regions, columns = years)
import numpy as np
malaria_matrix = np.array([[150, 200, 250, 300], [120, 180, 220, 260]])

üîç Explanation:

‚úî np.array() creates a NumPy array with rows and columns
‚úî malaria_matrix holds the malaria case data in a 2x4 format

Step 3: Dictionaries

# Creating a dictionary of malaria data
malaria_dict = {
"cases": cases_per_region,
"matrix": malaria_matrix,
"region": "West"
}

üîç Explanation:

‚úî {} creates a dictionary, which can store multiple types of data with named keys
‚úî malaria_dict contains the list, matrix, and a text string "West"

Step 4: DataFrames

# Creating a DataFrame for malaria cases per region
import pandas as pd
malaria_df = pd.DataFrame({
"region": ["West", "East", "North", "South"],
"cases": cases_per_region
})

üîç Explanation:

‚úî pd.DataFrame() creates a DataFrame, which is used to store tabular data
‚úî malaria_df is a DataFrame where region is a column of text and cases is a column of numbers

üîç Explanation:

1Ô∏è‚É£ Data Cleaning and Preprocessing

Exercise 1.1: Handle Missing Values

# Find the number of missing values in the 'Cases' column
missing_values = malaria_data['Cases'].isna().sum()
print(missing_values)

# Replace missing values with the mean of the 'Cases' column
mean_cases = malaria_data['Cases'].mean()
malaria_data['Cases'].fillna(mean_cases, inplace=True)

Explanation:

- isna() checks for missing values in the Cases column.
- mean() calculates the mean of the Cases column, automatically excluding missing values.
- fillna() with inplace=True replaces missing values directly in the DataFrame.

Exercise 1.2: Remove Rows with Missing Data

# Remove rows with any missing values using dropna()
malaria_data_cleaned = malaria_data.dropna()

# Display the cleaned DataFrame
malaria_data_cleaned.head()

Explanation:

- dropna() removes rows with any missing values in the entire DataFrame.

Exercise 1.3: Handle Outliers

# Calculate Q1, Q3, and IQR for the 'Cases' column
Q1 = malaria_data['Cases'].quantile(0.25)
Q3 = malaria_data['Cases'].quantile(0.75)
IQR = Q3 - Q1

# Calculate the lower and upper bounds for outlier detection
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

# Remove outliers from the 'Cases' column
malaria_data_no_outliers = malaria_data[(malaria_data['Cases'] >= lower_bound) & (malaria_data['Cases'] <= upper_bound)]

# Display the cleaned dataset
malaria_data_no_outliers.head()

Explanation:

- quantile() computes the first (Q1) and third (Q3) quartiles.
- IQR is calculated as the difference between Q3 and Q1.
- Outliers are removed based on the lower and upper bounds, defined by 1.5 times the IQR.

4Ô∏è‚É£ Bonus Challenge: Automate Data Cleaning and Visualization

# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

# Ask the user for the file path
file_path = input("Enter the path to the CSV file: ")

# Try to load the dataset and perform data cleaning and visualization
try:
# Load the dataset
malaria_data = pd.read_csv(file_path)

# Display basic information about the dataset
print("Dataset Information:")
print(malaria_data.info())
print("\nSummary Statistics:")
print(malaria_data.describe())

# Clean the data by handling missing values
# Replace missing values in 'Cases' with the mean
mean_cases = malaria_data['Cases'].mean()
malaria_data['Cases'].fillna(mean_cases, inplace=True)

# Handle outliers in the 'Cases' column using IQR
Q1 = malaria_data['Cases'].quantile(0.25)
Q3 = malaria_data['Cases'].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR
malaria_data = malaria_data[(malaria_data['Cases'] >= lower_bound) & (malaria_data['Cases'] <= upper_bound)]

# Create visualizations
# Create a histogram of 'Cases'
plt.figure(figsize=(10, 6))
plt.hist(malaria_data['Cases'], bins=10, color='blue', edgecolor='black')
plt.title('Distribution of Malaria Cases')
plt.xlabel('Cases')
plt.ylabel('Frequency')
plt.grid(True, linestyle='--', alpha=0.7)
plt.savefig('malaria_cases_histogram.png')
plt.close()

# Create a scatter plot of 'Cases' vs 'Fatalities'
plt.figure(figsize=(10, 6))
plt.scatter(malaria_data['Cases'], malaria_data['Fatalities'], color='red', alpha=0.7)
plt.title('Malaria Cases vs Fatalities')
plt.xlabel('Cases')
plt.ylabel('Fatalities')
plt.grid(True, linestyle='--', alpha=0.7)
plt.savefig('malaria_cases_vs_fatalities.png')
plt.close()

# Save the cleaned dataset to a new CSV file
clean_file_path = "cleaned_" + file_path.split("/")[-1]
malaria_data.to_csv(clean_file_path, index=False)
print(f"\nCleaned data saved to {clean_file_path}")
print(f"Visualizations saved as PNG files")

except FileNotFoundError:
print(f"Error: The file {file_path} was not found.")
except Exception as e:
print(f"An error occurred: {e}")

Explanation:

- The script asks the user for a file path and loads the dataset using pd.read_csv().
- It handles missing values using fillna() and removes outliers based on the IQR method.
- It creates a histogram and scatter plot for visualizations and saves them as PNG files.
- Finally, it saves the cleaned dataset using to_csv().
- Error handling is implemented using try-except blocks.
